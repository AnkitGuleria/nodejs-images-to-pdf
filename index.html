Install GraphicsMagick

Instructions
  1. Download GraphicsMagick
  2. Download jpeg library
  3. Download png library
  4. In lib directories run Command: ./configure && make && make install
  5. Command to unzip: gzip -d GraphicsMagick-1.3.tar.gz | tar -xvf -
  6. Command: cd GraphicsMagick-1.3
  7. Command: ./configure CC=clang
  8. Commnad: make
  9. Command: make install
  10. Command to verify: gm display
